Module Udns.Cname

Canonical name

A canonical name (CNAME) is an alias. host.example.com CNAME foo.com redirects all record sets of host.example.com to foo.com.

type t = Domain_name.t

The type of a canonical name.

compare a b compares the canonical name a and b.
